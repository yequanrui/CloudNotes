> 常见目录

| 目录                    | 说明                       |
| :---------------------- | :------------------------- |
| `/`                     | root用户根目录             |
| `/root`                 | root用户目录               |
| `/home`                 | 普通用户根目录             |
| `/home/{username}`      | 普通用户目录               |
| `/etc`                  | 配置文件目录               |
| `/sbin`                 | root用户管理命令和程序目录 |
| `/bin`                  | 普通用户管理命令和程序目录 |
| `/usr/sbin`、`/usr/bin` | 系统预装的其他命令目录     |
| `/usr/local`            | 用户安装的程序目录         |
| `/opt`                  | 第三方安装的程序目录       |
