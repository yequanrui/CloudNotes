### 微信隐藏技巧

1. 恢复聊天记录：微信搜索“recover”一微信修复工具故障修复 一 聊天记录 — 确认
2. 建单人群：添加好友一面对面建群—随机输入4个数字，即可建立一个单人群，想怎么嗨就怎么嗨~
3. 判断删除的好友：向目标好友转账任意金额，若对方已删除了你，则会提示非好友，如你是对方好友，则会要求输入支付密码。不输入密码就行了。
4. 备忘录置顶提醒：我一收藏一右上角的“+”一写完备忘信息后点击右上方“…”，选择“在聊天中置顶”，然后每次打开微信就可以看到置顶提醒了，支持添加语音、位置和图片
5. 重置微信二维码：个人信息一我的二维码右上角的“…”一重置二维码。
6. 聊天视频静音播放：长按视频两秒选择【静音播放】就可以无声观看视频啦，朋友圈视频也可以这样操作。
7. 拉黑朋友圈：长按头像—设置权限选择【仅聊天】或【不看她】、【不让她看】以后对方更新的朋友圈消息就都不会出现在你的朋友圈里了。
8. 快速查看未读消息：双击左下角“微信”
9. 找回失踪的群聊：发起群聊一选择一个群，可以看到所有群。
10. 隐藏聊天记录（i0S14）：选中想隐藏的聊天会话，向左滑动会发现新增了“不显示”选项，点击后该会话将从首页隐藏但，聊天记录不会删除，有新消息时会重新显示
11. 开启青少年模式（i0S14）：iOS用户更新到7.0.17最新版本，前往“我”“设置”便可开启青少年模式，家长也可根据具体情况设置允许访问的范围
12. 信息提醒功能：长按信息（要提醒的事）-提醒-设置提醒时
13. 简单拼接长图：在收藏界面添加一个新的笔记，然后把你需要拼接的图片按照顺序放进去，就能保存成长图，图片加文字也可以。
14. 语音分享朋友圈：打开微信"我"→点开收藏→点击右上角的加号→点击话筒开始录制→录制完成后点击话筒的图标，点击右上角的三个点分享到朋友圈。
15. 快速调出付款码：平时买东西发现还要点击自己的码特别麻烦，后来发现其实在桌面长按微信图标即可快速出示付款码。
16. 删除登录记录：不仅能避免侦探女友查岗，还能避免账号被盗风险微信【设置】→【账号安全】→【删除登录记录】→完成
17. 悄悄加回已删好友：可以通过群聊加回，也就是群里有她在的，这样只有单方面通知，可以避免好友验证的尴尬
18. 屏蔽某条朋友圈评论提醒：点击你朋友圈的“消息列表”→选择你看得到的某条评论→左滑选择“不再通知”，然后这条朋友圈评论的消息就不再提醒了
19. 红包捡漏：打开有发红包的群，点击左上角的\[→查找聊天记录→选择\[交易\]，没有显示“已被领完”的说明红包还可以领哦
20. 钱包加密：我们完全可以通过设置，给钱包\]加密。打开我的界面，依次点击「支付」→「钱包」→「安全保障」→「安全锁」。
21. 分组屏蔽朋友圈：点开标签，新建一个标签，点击右上角的导入，再选择你们想要屏蔽的人群，同事亲戚之类的，分组好下次屏蔽直接按分组就可以啦！
22. 电子身份证：点击微信搜一搜→输入“CTID”→打开小程序→注册账号→领取身份证>之后在卡包中可以直接打开
23. 文字重新编辑：发信息打错字的时候，选择撤回→点击重新编辑→再发送就可以。是不是简单快捷，也省去了解释的麻烦了。
24. 朋友圈静音：朋友圈视频静音播放，我们在某些不方便的时候，遇到想看的朋友圈视频，可以长按视频，选择静音播放。这样就不会给身边其他人听见了

### 解决“请在微信客户端中打开链接”

在浏览器访问网站，却提示“请在微信客户端打开链接”。

虽然这个情况你可能从未遇到过，但对于爱折腾的小伙伴，确是一道拦路虎

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/p69qCYb67zqwDQgTxFv6H713bjh8DoeMusvB2g4zGAviaFAvyyVIWL32Rfqk0GcPiaW8JW3x3hFwScmiaepdaibVkg/640?wx_fmt=jpeg&wxfrom=5&wx_lazy=1&wx_co=1)

其实解决办法很简单，就是新建一个UA：

按F12打开浏览器的控制台，切换到手机模式（点亮下图所指蓝色图标）。

![图片](https://mmbiz.qpic.cn/mmbiz_png/p69qCYb67zqwDQgTxFv6H713bjh8DoeMuYrnMDsUkFpSoGnn8vUc1OiacwaCXkDUu7vZgyfu7gVr0nPqkvnDyPg/640?wx_fmt=png&wxfrom=5&wx_lazy=1&wx_co=1)

选择“Edit...”后点击“Add custom device...”。

![图片](https://mmbiz.qpic.cn/mmbiz_png/p69qCYb67zqwDQgTxFv6H713bjh8DoeMRzEpjnVaP5VCI9Jx294s59XumicJtUbMzoHzIOavH3GShCsEyFnGic3A/640?wx_fmt=png&wxfrom=5&wx_lazy=1&wx_co=1)

![图片](https://mmbiz.qpic.cn/mmbiz_png/p69qCYb67zqwDQgTxFv6H713bjh8DoeMTAJyCkxcZMdMpjwOVlqsSQa8jN29ib0yngTjvNkYviaDvHibDj7CkXzcA/640?wx_fmt=png&wxfrom=5&wx_lazy=1&wx_co=1)

User agent string 输入以下代码：

```text
Mozilla/5.0 (Linux; Android 5.0; SM-N9100 Build/LRX21V) > AppleWebKit/537.36 (KHTML, like Gecko) Version/4.0 > Chrome/37.0.0.0 Mobile Safari/537.36 > MicroMessenger/6.0.2.56_r958800.520 NetType/WIFI
```

保存后将需要绕过限制的链接使用此UA访问即可。
